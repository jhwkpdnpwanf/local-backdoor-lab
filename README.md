# 로컬 환경에서 백도어 실습

> 이 프로젝트는 오로지 보안 학습 목적입니다.  
> 모든 실습은 로컬 환경(VMware)에서만 실행되며, 실제 외부 시스템을 대상으로 실행하지 않습니다.  
> 학습 용도로만 사용해주세요.
<br>

### 학습 환경  
* **가상화 환경**: VMware Workstation Pro
* **공격 대상 환경**: Kali Linux (VM)
* **호스트 OS**: Windows 11 (WSL 사용)
<br>
